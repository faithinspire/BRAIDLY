🎯 WHAT HAPPENS BEHIND THE SCENES:

1. Click "Create Demo Users"
   ↓
   JavaScript loads existing users from localStorage
   ↓
   Checks if demo users already exist
   ↓
   Creates demo users if they don't exist
   ↓
   Saves to localStorage
   ↓
   Shows success message

2. Click "Login as customer"
   ↓
   JavaScript calls login(email, password)
   ↓
   Looks up user in mock database
   ↓
   Verifies password
   ↓
   Stores in localStorage
   ↓
   Redirects to dashboard
