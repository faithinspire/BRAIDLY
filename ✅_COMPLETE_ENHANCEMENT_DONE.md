### Customer Flow
1. **Signup** → Select "Customer" role
2. **Login** → Redirects to Customer Dashboard
3. **Browse Braiders** → View available braiders
4. **Chat** → Message braiders
5. **Bookings** → Create and manage bookings
6. **Profile** → Update profile information

### Braider Flow
1. **Signup** → Select "Braider" role
