TROUBLESHOOTING
================================================================================

If you encounter issues:

1. Check browser console (F12) for error messages
2. Check Supabase logs (Dashboard → Logs)
3. Verify all tables exist (Dashboard → Table Editor)
4. Verify storage buckets exist (Dashboard → Storage)
5. Verify RLS policies are enabled (Dashboard → Authentication → Policies)
6. Run verification commands (🔧_VERIFICATION_COMMANDS.sql)

Common Issues:

ISSUE: Schema deployment fails
FIX: Check you're in correct Supabase project, have admin access

ISSUE: Admin login fails
FIX: Check admin user exists in auth.users and profiles table

ISSUE: Signup → login fails
FIX: Check profile was created, wait 2-3 seconds after signup

ISSUE: Avatar upload fails
FIX: Check storage buckets exist, check storage policies

ISSUE: Portfolio upload fails
FIX: Check portfolio_images table exists, check storage policies

ISSUE: Earnings display shows error
FIX: Check bookings table exists, check RLS policies
