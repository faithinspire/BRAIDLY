🚀 HOW TO START
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Start Local Server
   
   Option A - Python:
   $ python -m http.server 8000
   
   Option B - Node.js:
   $ npx http-server -p 8000
   
   Option C - PHP:
   $ php -S localhost:8000
   
   Option D - Just double-click index.html

STEP 2: Open Browser
   
   Navigate to: http://localhost:8000

STEP 3: Login
   
   Click "Demo Access" in navigation
   OR go to: http://localhost:8000/welcome.html
   Copy credentials and login!

🐛 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Problem: Can't login
Solution: 
   • Use exact credentials (case-sensitive)
   • Try clearing browser cache
   • Use incognito mode

Problem: Images not showing
Solution:
   • Images are placeholders
   • Add real images to assets/images/ folder
   • See SETUP.md for image specs

Problem: Page looks broken
Solution:
   • Make sure you're using a local server
   • Don't open HTML files directly
   • Check browser console (F12) for errors

Problem: Photo editor not working
Solution:
   • Upload an image first
   • Try different image format (JPG, PNG)
   • Check browser console for errors
