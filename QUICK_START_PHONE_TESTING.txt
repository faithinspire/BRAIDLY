📱 INSTALL PWA ON PHONE:

Android (Chrome):
  1. Open app in Chrome
  2. Tap menu (⋮) → "Install app"
  3. Tap "Install"
  4. App on home screen ✅

iOS (Safari):
  1. Open app in Safari
  2. Tap Share (↗️)
  3. Scroll → "Add to Home Screen"
  4. Tap "Add"
  5. App on home screen ✅

🔧 TROUBLESHOOTING:

Problem: Can't reach server
Solution: 
  • Check phone is on same WiFi
  • Use exact IP from npm run dev
  • Check firewall allows Vite

Problem: WebSocket fails
Solution:
  • Already fixed! Check console
  • Restart dev server if needed

Problem: Signup loading forever
Solution:
  • Check browser console (F12)
  • Verify .env file has Supabase keys
  • Try on desktop first

Problem: PWA won't install
Solution:
  • Must be HTTPS (works on Vercel)
  • Try different browser
  • Clear browser cache
