HOW TO TEST:

1. START DEV SERVER
   npm run dev
   (Should run on http://localhost:5173)

2. TEST SIGNUP
   - Go to http://localhost:5173/signup
   - Fill form:
     * Full Name: John Doe
     * Email: john@example.com
     * Password: password123
     * Role: Customer
   - Click "Create Account"
   - Should redirect to dashboard

3. TEST LOGIN (SAME USER)
   - Go to http://localhost:5173/logout (or clear localStorage)
   - Go to http://localhost:5173/login
   - Fill form:
     * Email: john@example.com
     * Password: password123
   - Click "Sign In"
   - Should redirect to dashboard

4. TEST LOGIN (WRONG PASSWORD)
   - Go to http://localhost:5173/login
   - Fill form:
     * Email: john@example.com
     * Password: wrongpassword
   - Click "Sign In"
   - Should show error: "Invalid password"

5. TEST DUPLICATE SIGNUP
   - Go to http://localhost:5173/signup
   - Try to sign up with john@example.com again
   - Should show error: "Email already registered"
