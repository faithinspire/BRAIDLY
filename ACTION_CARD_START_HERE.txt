🎯 QUICK START (5 MINUTES):

  1. Open terminal in project folder
  
  2. Install dependencies:
     $ npm install
  
  3. Create .env file with Supabase credentials:
     VITE_SUPABASE_URL=https://your-project.supabase.co
     VITE_SUPABASE_ANON_KEY=your-anon-key
  
  4. Start development server:
     $ npm run dev
  
  5. You'll see:
     ➜  Local:   http://localhost:5173/
     ➜  Network: http://192.168.x.x:5173/
  
  6. Open on phone:
     • Connect phone to same WiFi
     • Open browser on phone
     • Paste the Network URL
     • App loads! ✅

🔧 TROUBLESHOOTING:

  Problem: Can't reach server from phone
  Solution: 
    • Check phone is on same WiFi
    • Use exact IP from npm run dev output
    • Check firewall allows Vite
  
  Problem: WebSocket fails
  Solution:
    • Already fixed! Check console
    • Restart dev server if needed
  
  Problem: Signup loading forever
  Solution:
    • Check browser console (F12)
    • Verify .env has Supabase keys
    • Try on desktop first
  
  Problem: PWA won't install
  Solution:
    • Must be HTTPS (works on Vercel)
    • Try different browser
    • Clear browser cache
