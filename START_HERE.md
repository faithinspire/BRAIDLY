## ⚡ Quick Start (3 Steps)

### Step 1: Setup Database (5 min)
```
1. Go to https://app.supabase.com
2. Select project: rsemdjxizhkqaoptdxlc
3. SQL Editor → New Query
4. Copy supabase/schema.sql
5. Paste and Run
6. Create 3 storage buckets: avatars, portfolio, gallery
```

### Step 2: Start Dev Server (1 min)
```bash
npm run dev
```

### Step 3: Create Account (2 min)
```
1. Visit http://localhost:5173
2. Click Sign Up
3. Fill in details
4. Done!
```

**Total Time**: ~10 minutes

## 🆘 Common Issues

### "Auth session missing" Error
- Normal when not logged in
- Just sign up with a new account

### "Database tables not found" Error
- Run `supabase/schema.sql` in Supabase SQL Editor
- Refresh the app

### "Storage bucket not found" Error
- Create 3 storage buckets: avatars, portfolio, gallery
- Make them public

### Still Having Issues?
- Check `TROUBLESHOOTING.md`
- Check browser console (F12)
- Restart dev server
