🔍 WHY THIS HAPPENED
═══════════════════════════════════════════════════════════════════════════════

The `-webkit-text-fill-color: transparent` property was intended to show a
gradient background through the text. However:

1. The gradient wasn't rendering properly
2. The text became completely invisible instead
3. No fallback color was set
4. The text was hidden on all browsers

This is a known CSS issue where `-webkit-text-fill-color: transparent` can
cause text to disappear if the background-clip gradient doesn't render.

The fix: Use explicit color instead of transparent text-fill-color.
