## 🚀 Quick Commands

### Start Server
```bash
# Python
python -m http.server 8000

# Node.js
npx http-server -p 8000

# PHP
php -S localhost:8000
```

### Access URLs
```
Landing:   http://localhost:8000/index.html
Welcome:   http://localhost:8000/welcome.html
Login:     http://localhost:8000/login.html
```
